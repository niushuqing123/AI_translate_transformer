# Outputs

This repository keeps textual experiment artifacts in `outputs/` so the training process, metrics, and report materials are preserved.

Kept in git:

- `*.md`
- `*.json`
- `*.jsonl`
- `*.txt`
- `*.csv`

Excluded from git:

- model checkpoints
- plot images
- temporary runner redirection logs

The most important summary files are:

- `outputs/final_workflow_summary.md`
- `outputs/baseline_tiny_en_de/run_summary.md`
- `outputs/experiments/summary.md`
